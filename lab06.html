Hometown Map

Miranda Grein | DCDA 40833

Introduction

This lab presents an interactive map highlighting meaningful locations from my hometown. I created the map using Python and Folium, and included markers with detailed descriptions.

Requirements

  • Choose a personally significant piece of work to inspire design choices for map
  • Create a custom Mapbox map style based on this piece and customize components toward this vision
  • Create a Hometown Locations dataset containing important locations from your hometown
  • Use this dataset to create an interactive map using Folium and Python
  • Reflect on design choices, challenges, and AI considerations

Design Inspiration

Design Inspiration

Claude Monet's The Marina at Argenteuil inspired my custom mapbox design. I used Monet's palette with soft blues, greens and gentle warm earth tones. In my basemap, the base water tones come from Monet's soft blues and the land layers mixed different shades of Monet's greens. The labels, text, and accents use the muted colors mixed in the painting. Monet uses soft brush strokes and blended edges in his paintings which I tried to recreate by using low opacities and gentle colors.

Reflection

Cartographic Challenges

The main challenge I faced when designing my basemap for multiple scales was ensuring the opacity for the roads did not get too high, so that the text on the roads could still be seen as the text was a lighter color. I had to experiment with the opacity and color as the text font was harder to see in certain combinations. I decided to show very few roads at a higher zoom level so the land and outlines were emphasized. Zooming in more showed more secondary roads and detail-orientated aspects of the map. This was done to especially to slowly introduce more details as too much detail at first proved to have too much noise.

Technical Challenges

I encountered some challenges while using Python to create the interactive map. First, the csv file had inconsistent whitespaces and quotes which I had to fix. I was able to use AI as a tool to identify that this was an issue and clean the csv in a way which would fix the issue. Coding in Python with a new package and using mapbox for the first time caused syntax issues, especially when nesting expressions. Even small formatting errors would prevent the map from rendering correctly. AI tools were helpful for checking the structure, suggesting new code, explaining error messages and adjusting code, and providing corrected formatting examples.
