Hometown Map

This lab presents an interactive map highlighting meaningful locations from my hometown. I created the map using Python and Folium, and included markers with short personal reflections.

The interactive Folium map was exported as an HTML file and can be viewed locally or embedded in a notebook. If you want to explore the map interactively, open the exported HTML in a browser or visit the linked notebook (if provided).

How to view the interactive map

  1. If you have the exported Folium HTML file, open it directly in your browser.
  2. Or, run the Python notebook that generates the map (Folium) and display it inline.
  3. If you'd like, I can add an embedded iframe of the map here or include the exported HTML file in the repo — tell me which you prefer.

Notes: If the interactive map isn't included, the static preview above gives an idea of layout and marker placement. Let me know if you'd like responsive styling adjustments or an embedded interactive map.
